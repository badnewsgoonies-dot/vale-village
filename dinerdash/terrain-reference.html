Currently these are sprite sheets (PNG files with multiple tiles). To use individual tiles:
  1. Open the sprite sheet in an image editor
  2. Extract individual 16x16 or 32x32 tiles you want
  3. Save them to assets/terrain/
  4. Run node scripts/generate-sprite-list.js
  5. Reload the app to see your new terrain tiles!
