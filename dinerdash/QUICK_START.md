## How to Use

1. **Browse sprites** in the left sidebar by category
2. **Drag sprites** onto the canvas or double-click to add
3. **Click and drag** placed sprites to move them
4. **Right-click and drag** to pan the view
5. **Scroll wheel** to zoom in/out
6. **Press Delete** to remove selected sprite
7. **Export PNG** button saves your mockup as an image
8. **Save Layout** exports positions for later use

## Adding More Sprites

Want to add more sprites from vale-village?

1. Copy the sprite files to the appropriate `assets/` folder
2. Edit `src/SpriteManager.ts` and add entries to `getSpriteList()`
3. The sprites will appear automatically when you reload
